# CS210-Airgead-Banking

Project Summary

This project is an investment calculator created for Airgead Banking. The goal of the program is to help users understand how their investments grow over time using compound interest. The program takes user input such as initial investment, monthly deposit, interest rate, and number of years, then generates two reports: one without monthly deposits and one with monthly deposits. This allows users to compare how consistent contributions impact long-term growth.

What I Did Well

One thing I did well in this project was organizing the program using object-oriented programming. I created an InvestmentCalculator class to handle all calculations and logic, which made the code more structured and easier to manage. I also implemented input validation to make sure users enter correct values, which helps prevent runtime errors and improves the overall reliability of the program.

Areas for Improvement

One area I could improve is making the program more flexible and user-friendly. For example, I could allow users to run multiple calculations without restarting the program or add options to save results to a file. I could also improve efficiency by optimizing calculations or expanding the program to handle more advanced financial scenarios. These improvements would make the program more practical and scalable.

Most Challenging Parts

The most challenging part of this project was handling user input and making sure invalid inputs didn’t break the program. I had to carefully use error handling and input validation to catch issues like non-numeric values or negative numbers. I overcame this by testing different inputs and using resources like zyBooks and debugging tools to fix problems step by step.

I made this program maintainable and readable by using clear function names, organizing the code into separate files, and keeping the main function simple. The use of a class makes it easy to update or expand the program in the future without rewriting everything. This structure also makes the code easier for others to understand and modify.
